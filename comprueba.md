# comprueba (M) #

Devuelve True si la matriz M está correctamente formateada para su uso con el módulo, y False en caso contrario.

## Ejemplos ##
```
>>> import matrices
>>> A = [[1,2,3,4,5,6,7,8,9], [3,3]]
>>> matrices.comprueba(A)

  True

>>> B = [[1,2,3,4,5,6,7,8,9], [2,2]]
>>> matrices.comprueba(B)

 False
```
